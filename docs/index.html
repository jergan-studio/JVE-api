Welcome

JVE API is a lightweight scripting environment powered by JIV Core Commands. It provides alert systems, canvas rendering, tabs, and interactive scripting.

Getting Started

<script src="jiv.core.js"></script>
<script src="jve.api.js"></script>

After loading the runtime, JIV commands become available.

Core Commands

Alert()

Displays a browser alert.

Alert(browser)("Hello World")

NewTab()

Creates a new canvas tab.

NewTab(canvas)

Canvas()

Draws objects on a canvas.

Canvas(draw, rect 50 50 100 100)

Interactive.JVE

Main script format used by JVE API.

Alert(browser) Hello JVE

NewTab(canvas)

Canvas(draw, rect 50 50 100 100)

Canvas(draw, text Hello World)

Example

Alert(browser)("Welcome")

NewTab(canvas)

Canvas(draw, clear)
