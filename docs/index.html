JVE API

Jergan Verse Environment — Scripting & Command System

What is JVE?

JVE API is a lightweight scripting system powered by JIV Core Commands. It is designed for simple scripting, canvas drawing, and interactive apps.

Core Object: JIV

JIV is the main command API used inside JVE.

JIV.Alert("browser")("Hello World")
JIV.NewTab("canvas")
JIV.Canvas("draw", "rect 10 10 100 100")

Interactive.JVE Example

Alert(browser) Hello JVE

NewTab(canvas)

Canvas(draw, rect 50 50 120 80)

Canvas(draw, text Welcome)

Features

  • Alert System
  • Canvas Drawing Engine
  • Window / Tab System
  • Simple Script Format (.JVE)
